# Contributing to vMUD

Thanks for your interest in vMUD.

## Source Code

vMUD is a proprietary project. The source code is not publicly available,
and external code contributions (pull requests) are not accepted.

## Feedback and Bug Reports

If you find a bug or have a suggestion, feel free to open an issue.
Please include:

- A clear description of the problem
- Steps to reproduce (if applicable)
- Screenshots or logs if helpful

## Feature Requests

Feature ideas are welcome, but there is no guarantee they will be implemented.

## Conduct

Please keep discussions respectful and constructive. Personal attacks,
harassment, or bad-faith behavior will be ignored or removed.

## Security

If you believe you have found a security issue, please do not post it publicly.
Contact me directly instead.

---

vMUD is built and maintained as a personal project for the MUD community.
